# pic-selector 需求文档

## 1. 项目概述

**目标**

- 为摄影师/内容创作者提供一套本地化图片管理与智能筛选工具。
- 支持批量导入、缩略图优化、AI 识别分析与多维度分类，提升选片效率。

**范围**

- 本地图库管理与文件夹切换
- 图片缩略图生成与缓存
- AI 分析（人脸、风格、影调、颜色、质量、重复等）
- 分类与筛选、排序、评分、全屏预览
- 结果持久化与后续复用（含人物命名与学习）

## 2. 用户画像与使用场景

- 摄影师：大量照片快速筛选与评分
- 设计/内容团队：多图库管理与分类检索
- 个人用户：自建相册与人物/风格整理

典型流程：
1. 导入文件夹或拖拽进入图库
2. 自动按扩展名分类并默认进入 JPG（或 RAW）子目录
3. 生成并持久化缩略图
4. 批量 AI 分析，按维度分类
5. 筛选、排序、评分、全屏预览
6. 持久化保存并复用结果

## 3. 功能需求（结构化）

### 3.1 图库与文件管理
- 支持多文件夹图库管理
- 图库与文件夹页面分离
- 支持拖拽文件夹进入图库
- 导入时执行“按扩展名分组”逻辑
  - 以图片扩展名分组，创建文件夹并将图片放置在对应扩展名的文件夹中
  - 默认导入 JPG/JPEG 子目录
  - 若无 JPG/JPEG，则导入 RAW 子目录（DNG/RWA/ARW/CR2/CR3/NEF/RW2/ORF/RAF/SR2/SRW/PEF/3FR 等）
  
- 支持图库卡片菜单（删除、移除等）
- 鼠标悬浮点击缩略图进入文件夹
- 分析指标可配置

### 3.2 缩略图与性能
- 操作界面仅加载缩略图，原图仅在全屏预览加载
- 支持缩略图本地缓存与持久化
- 多线程生成缩略图
- RAW/DNG 导入稳定性：避免 `System.OutOfMemoryException`
- 缩略图显示适配：每排数量可调（默认 5，最大 10）
- 滚动条与布局自适应

### 3.3 AI 分析与分类
- 人脸检测与计数：有人脸/多人/无人脸
- 人物识别与分组
  - 支持人物名称修改
  - 同名自动合并
  - 人物标注持久化与自主学习，需要本地模型
- 风格识别与分类
- 颜色识别与分类
  - 颜色分析改为“高光/阴影/中间调”多维分析
  - 当黑白灰占比过高时，仍优先显示其它主色
- 质量评估与废片判断
- 评分系统（1–5星）

### 3.4 分类列表
- 分类维度：
  - 全部
  - 人脸（有人脸/多人/无人脸/人物）
  - 质量（过曝/过暗/废片/可用/未分析/Top10%）
  - 星级（1-5）
  - 影调
  - 风格（高对比，浓郁，冷酷，高饱和）
  - 颜色（带色块标识）
  - 设备（相机品牌/型号）
- 分类列表支持右键菜单：归档（合并到其他分类）
- 分类列表保持展开状态，不因刷新自动折叠

### 3.5 预览与交互
- 全屏预览支持：缩放、拖拽、左右切换、点击退出
- 全屏预览显示：主色圆点、星级评分按钮
- 缩略图支持右键菜单与悬浮菜单按钮
- 缩略图支持人物标记、清除/添加（下拉选择）

### 3.6 UI/UX
- 全局风格统一（WPF UI/Flutter风格组件）
- 圆角矩形边框为主
- 颜色识别项左侧显示圆点，其它项不显示
- 启动加载动画 + 渐入主界面
- 长操作显示 Loading，支持取消
- 详情面板可折叠，按钮位置合理不占位
- 滚动条细窄、圆角、方向自然

### 3.7 数据持久化
- 保存分析结果、分类与人物命名
- 重进文件夹自动加载历史结果
- 人物命名可跨图库复用
- 支持归档映射持久化

## 4. 非功能需求

- **性能**：缩略图优先、并行生成、内存可控
- **稳定性**：RAW 文件不崩溃、异常可控
- **可维护性**：UI 组件拆分、代码模块化
- **可扩展性**：菜单功能可扩展

## 5. 技术栈与实现

### 客户端
- 平台：Windows
- UI：WPF (.NET 8)
- 数据持久化：SQLite，可扩展为MySql
- 图片处理：System.Drawing + WPF BitmapImage，为我推荐更优方案
- 滚动/控件样式：自定义 ResourceDictionary

### AI 服务
- Python AI 服务（python-ai-service）
- 模型/插件式分析：人脸、风格、影调、颜色、质量
- HTTP 调用：客户端通过 HttpAiServiceClient

## 6. 关键约束
- UI 必须统一风格（控件本身定制，不依赖外部卡片包裹）
- 缩略图与色卡必须在卡片内严格对齐，不能溢出
- 颜色识别维度必须稳定可用
- 人物识别必须支持长期学习和复用

## 7. 优先级

- P0：人脸识别问题修复、分类算法优化、数据持久化、窗口行为
- P1：图库系统、全屏预览、缩略图加载优化
- P2：评分/撤回、UI 动画、表单化显示
